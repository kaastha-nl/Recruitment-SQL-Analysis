# Recruitment Funnel Analysis (SQL)

This project analyzes recruitment data using SQL to understand hiring outcomes
and candidate decision patterns across companies and attributes.

## Objective
- Calculate overall and segmented hiring rates
- Analyze recruitment funnel outcomes
- Identify patterns in candidate selection decisions

## Dataset
The dataset contains candidate demographics, education, experience indicators,
company assignments, and final hiring decisions.

## Key Analysis
- Overall hiring rate calculation
- Hiring rate by company
- Hiring outcomes by education level and programming experience
- Gender-based hiring rate comparison

## Tools Used
- SQL (MySQL-compatible syntax)

## Repository Structure
- `sql/` – SQL queries for analysis
- `data/` – recruitment dataset (CSV)
